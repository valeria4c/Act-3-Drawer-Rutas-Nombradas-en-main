# myapp
Widget Reutilizable: En lugar de escribir el mismo código 5 veces, creé la función tarjetaEstudio. Esto hace que tu app sea más rápida y fácil de mantener.

BoxShadow: Añadí una sombra sutil para darle ese efecto de profundidad que se ve en tu imagen original.

Row & Column: Usé una fila (Row) para separar la imagen del texto, y una columna (Column) interna para que el nombre del negocio esté debajo del nombre del servicio.

Responsive: Al usar un Container con anchos definidos para la imagen, te aseguras de que se vea bien en diferentes tamaños de pantalla.
